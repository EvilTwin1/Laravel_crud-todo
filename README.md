# Instalation
```
 docker compose up
 composer install
 php artisan migrate
 php artisan db:seed
```
## Design feature

- Реализация валидации форм
- Реализация CRUD
- Реализация уведомлений о добавлении/удалении
- Реализация фильтрации по критерию
- Реализация кастомных ошибок
